## 📘 Course Overview

**DSBDA (Data Science and Big Data Analytics)** focuses on:

* Big Data processing using **Hadoop ecosystem**
* Data Analytics using **Python**
* Data Visualization using **Matplotlib, Seaborn, and Tableau**
* Distributed computing concepts like **MapReduce & Hive**

This repository contains practical assignments divided into:

* **Group A – Hadoop Based Assignments**
* **Group B – Data Analytics Using Python**

# 🔹 Group A – Assignments Based on Hadoop

## 1️⃣ Hadoop Installation

### Tasks:

* Single Node Hadoop Installation
* Multi Node Hadoop Installation

### Tools & Technologies:

* **Hadoop Distributed File System (HDFS)**
* **YARN**
* Java
* Linux (Ubuntu)

### Expected Outcome:

Successful configuration and execution of Hadoop in pseudo and distributed mode.

---

## 2️⃣ MapReduce Distributed Application (Java)

### Objective:

Design a distributed application using **MapReduce** to process a system log file.

### Problem Statement:

* Process a log file.
* Identify users who logged in for the **maximum duration**.
* Use a simple internet log file.
* Run in pseudo distributed Hadoop mode.

### Concepts Covered:

* Mapper & Reducer
* Key-Value Pair Processing
* Log Parsing
* HDFS File Handling

---

## 3️⃣ HiveQL – Flight Information System

### Tasks:

✔ Creating, Dropping, and Altering Database Tables
✔ Creating External Hive Table
✔ Load Data into Table
✔ Insert New Records
✔ Perform Joins
✔ Create Index
✔ Find Average Departure Delay per Day in 2008

### Tools Used:

* Hive
* HDFS
* SQL-like HiveQL

### Learning Outcome:

Hands-on experience with **Data Warehousing using Hive**.
